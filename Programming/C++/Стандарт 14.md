### Binary literals

Binary literals provide a convenient way to represent a base-2 number. It is possible to separate digits withÂ `'`.

```c
0b110 // == 6
0b1111'1111 // == 255
```

### Variable templates

C++14 allows variables to be templated:

```c
template<class T>
constexpr T pi = T(3.1415926535897932385);
template<class T>
constexpr T e  = T(2.7182818284590452353);
```
