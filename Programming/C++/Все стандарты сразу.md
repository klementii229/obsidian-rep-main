## C++11
### Основные нововведения:

1. **Автоматический вывод типов (`auto`)**: 
```cpp
    auto x = 10; // x будет int 
```

2. **Range-based for loops**:
```cpp
for (auto& item : container) { ... }
```

3. **Лямбда-выражения**:
```cpp
auto func = [](int x) { return x * 2; };
```

4. **Умные указатели (`std::unique_ptr`, `std::shared_ptr`, `std::weak_ptr`)**:
```cpp
std::unique_ptr<int> ptr = std::make_unique<int>(10);
```

5. **Правые ссылки и семантика перемещения (`std::move`, `std::forward`)**:

```cpp
    std::vector<int> v1 = {1, 2, 3};
    std::vector<int> v2 = std::move(v1); // Перемещение
```

6. **Новые контейнеры**:
- `std::array`
- `std::unordered_map`
- `std::unordered_set`
    
7. **Многопоточность** 
- `std::thread`
- `std::mutex`
- `std::condition_variable`
    
8. **Ключевое слово `nullptr`**:
```cpp
	int* ptr = nullptr;
```

9. **Делегирующие конструкторы**:
```cpp
    class MyClass {
    public:
        MyClass(int x) : MyClass(x, 0) {} // Делегирование
        MyClass(int x, int y) { ... }
    };
```

10. **Инициализация в стиле списка**:
```cpp
std::vector<int> v = {1, 2, 3};
```

11. **`constexpr`**:  
```cpp
constexpr int square(int x) { return x * x; }
```

12. **`std::function`**:  
```cpp 
std::function<int(int)> func = [](int x) { return x * 2; };
```

13. **`std::initializer_list`**:  
```cpp
std::vector<int> v = {1, 2, 3};
```    

14. **`std::to_string`**:  
```cpp
std::string s = std::to_string(42);
``` 

15. **`std::async`**:  
```cpp
auto future = std::async([]() { return 42; });
```
### Добавленные заголовочные файлы:

- [ ] `<atomic>`: Атомарные операции.
    
- [ ] `<chrono>`: Работа со временем.
    
- [ ] `<condition_variable>`: Условные переменные для многопоточности.
    
- [ ] `<forward_list>`: Односвязный список.
    
- [ ] `<future>`: Асинхронные операции.
    
- [ ] `<initializer_list>`: Поддержка инициализации списком.
    
- [ ] `<mutex>`: Мьютексы.
    
- [ ] `<random>`: Генерация случайных чисел.
    
- [ ] `<ratio>`: Рациональные числа.
    
- [ ] `<regex>`: Регулярные выражения.
    
- [ ] `<scoped_allocator>`: Аллокаторы с областью видимости.
    
- [ ] `<system_error>`: Обработка системных ошибок.
    
- [ ] `<thread>`: Потоки.
    
- [ ] `<tuple>`: Кортежи.
    
- [ ] `<type_traits>`: Метапрограммирование.
    
- [ ] `<typeindex>`: Индексация типов.
    
- [ ] `<unordered_map>`: Хэш-таблицы (ассоциативные массивы).
    
- [ ] `<unordered_set>`: Хэш-таблицы (множества).


```cpp

```    

### Добавленные заголовочные файлы:
